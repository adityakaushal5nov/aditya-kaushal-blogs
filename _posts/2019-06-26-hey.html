Get all Fields for any SObject via this method

Map schemaMap = Schema.getGlobalDescribe();
Schema.SObjectType objectSchema = schemaMap.get(objName);
Map fieldsMap = objectSchema.getDescribe().getFields.getMap();
Set fieldsList = fieldsMap.keySet();
